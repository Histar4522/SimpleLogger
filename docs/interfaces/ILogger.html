    Interface ILogger

    A Logger instance. Call any of the log, warn, error, debug to write a line of log.

    To create a ILogger instance, call default.create. This method may invoke a synchronous filesystem write call to create a log file.

    Note that every method accepts any amount of parameters. Only the last parameter will be recorded as log, and the rest will be considered as context flags and surrounded by [].

    Objects and Errors is treated differently: util.inspect is called on an object to stringify it; error objects is converted to string using as Error.stack to show stack trace.

    Call getSubLogger to create another instance of ILogger, where context flags will be added automatically.

    Examples:

    const logger = Logger.create({writeToFile: true}).getSubLogger("GENERAL");
    logger.log("hello, this is a log");
    logger.error("hello, this is an error");
    logger.warn("hello, this is a warning");
    logger.debug("hello, this is a debug");
    logger.getSubLogger("OBJECT").log({key: "value", number: 1145, obj: {this: "is", nested: ["object", "!"]}});
    logger.getParentLogger().error(new Error("test error!"));
